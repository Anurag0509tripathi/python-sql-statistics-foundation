# Basic Statistics Notes

These are my Month 1 notes for basic statistics.

## Mean

Mean is the average of all values.

Formula:

```txt
Mean = Sum of all values / Number of values
Example:

Values = 10, 20, 30
Mean = 60 / 3 = 20
Median
Median is the middle value after sorting the data.

Example:

Values = 10, 20, 30
Median = 20
Mode
Mode is the value that appears most often.

Example:

Values = 10, 20, 20, 30
Mode = 20
Range
Range shows the spread between the highest and lowest value.

Formula:

Range = Maximum value - Minimum value
Example:

Values = 10, 20, 30
Range = 30 - 10 = 20
Why Statistics Matters in Data Science
Statistics helps in understanding data patterns, comparing groups, measuring uncertainty, and evaluating machine learning models.
